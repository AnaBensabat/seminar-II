How to Define Temperature in out of equilibrium systems

Ana Bensabat Paulino

What is temperature?

Temperature in equilibrium:

  • All temperature definitions are consistent
  • Different instruments measure the same temperature
  • Universal characteristic of thermodynamical systems
  • Reflects "hotness" and "coldness"

Caveats:

  • Only valid in systems in thermal equilibrium or locally in equilibrium
  • Some definitions depend on taking the thermodynamic limit

The standard definitions of temperature fall short for non-Hamiltonian, non-equilibrium and small systems.

Intermezzo: The fluctuation-dissipation theorem

The fluctuation-dissipation theorem (FDT) is a key principle in statistical mechanics that relates the fluctuations in a system's properties with the dissipation of energy in that system [3].

Preliminaries:

The dynamical response of an operator A w.r.t. an external perturbation is \[\begin{align} \delta\langle A(x,t)\rangle = \int\text{d}^dx'\text{d}t' \chi(x,x',t,t')\delta h(x',t')\quad \xrightarrow[\text{}]{\text{Fourier transform}} \quad x(\omega) = \chi(\omega)f(\omega), \\ \chi\equiv\text{response function} \end{align} \] It can be proven [3] that the imaginary part of the response function is proportional to the rate at which work done by external forces is dissipated as heat. The Fourier-space correlation function for two operators is given by: \[\begin{align} C_{AB}(\omega) = \langle A(\omega)B(\omega')\rangle\delta(\omega+\omega') &&\text{and} && S_{AB}(\omega) = C_{AB}(\omega) -\langle A\rangle\langle B\rangle 2\pi\delta(\omega). \end{align} \]

Back-of-the-envelope derivation for an harmonic oscillator:

Consider the equation of motion for a perturbed harmonic oscillator: \[\begin{align} \ddot{x} + \omega_0^2x + \gamma\dot{x} = f/m. \end{align} \] The response function can be obtained by taking the Fourier transform of the equation of motion: \[\begin{align} \chi(\omega) = \frac{1}{m}\frac{1}{-\omega^2+\omega_0^2-i\omega\gamma}. \end{align} \]
Considering the force is a noisy variable, \[\begin{align} f(\omega) = \xi(\omega)\quad \text{s.t.}\quad \langle\xi(t)\rangle = 0 \quad \text{and} \quad \langle \xi(t)\xi(t')\rangle = 2m\gamma k_BT\delta(t-t'). \end{align} \] we can write \[\begin{align} x(\omega) = \chi(\omega)\xi(\omega) = \frac{\chi(\omega)}{m[-\omega^2+\omega_0^2-i\omega\gamma]}. \end{align} \] It then follows \[\begin{align} C_{xx}(\omega) = 2m\gamma T |\chi(\omega)|^2. \end{align} \] Considering now the system is only subject to thermal noise, and hence, its average position is zero, \[\begin{align} S_{xx}(\omega) = \frac{2k_BT}{\omega} \text{Im}[\chi_{xx}(\omega)]^2. \end{align} \]
The thermal noise in a system is related to its ability to dissipate energy, and the strength of this noise can be predicted from the system's response to an external perturbation.

Active Brownian Particles (ABPs)

Active Brownian particles not only diffuse due to the collisions with the surrounding fluid, but they also self-propel.
\[ \begin{align} \gamma \mathbf{\dot{r}}_i& = F_\text{act}\mathbf{n}_i - \nabla_i \sum_{i\neq j}U(r_{ij}) + \sqrt{2\gamma k_B T}\xi_i \\ \mathbf{n}_i &= (\cos\theta_i(t),\sin\theta_i(t)) \quad \text{with} \quad \dot{\theta}=\sqrt{2D_{\theta_i}}\eta_i \quad \text{and} \quad D_\theta = 3\gamma k_BT/ \sigma^2_d \end{align} \]
\[ \begin{align} &\langle\xi_i(t)\rangle=0, \quad \langle\xi_i(t)\xi_j(t^\prime)\rangle=\delta_{ij}\delta(t-t^\prime)\\ &\langle\eta_i(t)\rangle=0, \quad \langle\eta_i(t)\eta_j(t^\prime)\rangle=\delta_{ij}\delta(t-t^\prime)\\ &(m=1) \end{align} \]
For the particle interaction we usually use a truncated and shifted Lennard-Jones potential:
\[ \begin{align} &U(r_{ij}) = \begin{cases} U_{LJ}(r_{ij})-U_{LJ}(r_0) && \text{for } r\leq r_0 \\ 0 && \text{for } r > r_0 \end{cases} \\ &U_{LJ}(r) = 4\epsilon\left[\left(\frac{\sigma}{r}\right)^{12}-\left(\frac{\sigma}{r}\right)^6\right] \end{align} \]
How do these structures - Mobility Induced Phase-Separations (MIPS) - emerge in the parameter space of: \[ \begin{align} \phi_P = N\frac{\pi \sigma_D^2}{4S} && \text{Pe} = \frac{Lv_\text{act}}{D} = \frac{\sigma_D F\text{act}}{k_B T} \end{align}\]

Conclusions

  • The non-trivial coupling between position and velocity in active systems leads to non-Boltzmann distributions making it impossible to uniquely quantify fluctuations based on a single temperature parameter.
  • The different temperature definitions can be grouped into two families.
  • The two families can be matched in the far-from equilibrium regime by rescaling temperatures with the particle mass.
